# Advanced Python Port Scanner

A multithreaded TCP port scanner built in Python for network reconnaissance and security testing.

## Features

- Multithreaded port scanning
- Banner grabbing
- Service detection
- Custom port ranges
- IP validation using IPy
- Fast concurrent scanning

## Technologies

Python  
Sockets  
Threading  
Queue  

## Example Output

[OPEN] Port 22 | SSH
Banner: SSH-2.0-OpenSSH

[OPEN] Port 80 | HTTP

## Usage

Install dependencies:

pip install -r requirements.txt

Run scanner:

python scanner.py
